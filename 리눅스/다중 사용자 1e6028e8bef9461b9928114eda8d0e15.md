# 다중 사용자

여러명이 운영체제를 같이 사용하는 방법

```bash
$ id
```

uid : 나의 계정 

$ who : 현재 누가 해당 서버에 접속해잇는지 보임 

User 2종류 

-super(root) user

-user

일반 유저가 명령어 앞에 sudo를 붙이면 root user로 실행하는 것 처럼 실행

 

슈퍼유저가 되는 방법

$ su : A라는 사용자가 B라는 사용자가 되고 싶을 때 or super유저가 되고싶을 때

슈퍼유저로 사용하는것보다 sudo를 이용해서 예외적으로 강력한 명령어를 적용하는 것이 좋다.
